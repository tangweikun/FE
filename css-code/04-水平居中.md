- 给 div 设置一个宽度，然后添加 `margin: 0 auto` 属性

```css
.center-horizontal {
  width: 200px;
  margin: 0 auto;
}
```

- 使用`flex`布局

```css
.center-horizontal {
  display: flex;
  justify-content: center;
}
```

- 使用`transform`

```css
.center-horizontal {
  position: relative;
  left: 50%;
  transform: translateX(-50%);
}
```
